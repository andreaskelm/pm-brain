# Focus Task

Priority assessment and context-switching management. Use Eisenhower Matrix, deep work principles, and LNO framework to maximize impact.

## Your Current Situation

Answer these first:
- How many initiatives/projects are you juggling?
- How much of your time is meetings vs. focused work?
- What's preventing you from working on what matters most?
- When was the last time you had 2+ hours of uninterrupted work?

## Eisenhower Matrix Assessment

Sort your tasks into four quadrants:

### QUADRANT I: Urgent & Important (DO NOW)
**Examples:**
- Production outages
- Critical bugs blocking customers
- Deadline today/tomorrow
- Crisis management

**Action:** Do immediately, but design systems to reduce these

### QUADRANT II: Not Urgent but Important (SCHEDULE FIRST)
**Examples:**
- Strategic planning
- Discovery/research
- Building relationships
- Process improvements
- Learning/growth

**Action:** Protect time for these! This is where real impact happens

### QUADRANT III: Urgent but Not Important (DELEGATE/MINIMIZE)
**Examples:**
- Most interruptions
- Some meetings
- Other people's priorities
- Email/Slack responses

**Action:** Delegate, set rules, time-box, or decline

### QUADRANT IV: Not Urgent & Not Important (DROP/AUTOMATE)
**Examples:**
- Busy work
- Vanity metrics
- Unnecessary reports
- Time wasters

**Action:** Stop doing these. Automate or eliminate

## Deep Work vs Shallow Work

### Deep Work (Protect This)
**Characteristics:**
- Requires focus and cognitive effort
- Creates reusable assets
- Changes decisions or outcomes
- Can't be done in meetings

**Examples:**
- Writing strategy docs
- PRDs and requirements
- Data analysis and synthesis
- Design thinking
- Complex problem solving

**Time needed:** 2-4 hour blocks

### Shallow Work (Batch This)
**Characteristics:**
- Coordination and communication
- Administrative tasks
- Can be interrupted
- Doesn't require deep thinking

**Examples:**
- Responding to Slack/email
- Status updates
- Quick questions
- Meeting attendance
- Calendar management

**Time needed:** 30-60 min batches

Reference: @02-Methods-and-Tools/2.0-Foundations/2.0.1-Mental-Models/4-Execution-Hygiene/1-deep-work-shallow-work.md

## LNO Framework: Leverage, Neutral, Overhead

### Leverage Activities (Maximize)
**What:** Work that creates disproportionate value
- Unblocking others
- Strategic decisions
- Process improvements
- Relationship building
- Multiplier effects

**Target:** 40-50% of time

### Neutral Activities (Optimize)
**What:** Necessary but doesn't multiply impact
- Routine meetings
- Status updates
- Standard execution work

**Target:** 30-40% of time

### Overhead Activities (Minimize)
**What:** Necessary evils, pure coordination
- Administrative work
- Reports
- Compliance tasks
- Calendar management

**Target:** <20% of time

## Common Scenarios

**"My calendar is 100% meetings"**
- Audit all meetings: which can be cancelled, async, or delegated?
- Block focus time 2 weeks in advance
- Set "no meeting" blocks (e.g., Tue/Thu mornings)
- Say no more: @02-Methods-and-Tools/2.4-Communication/2.4.6-Saying-No

**"Constant interruptions on Slack"**
- Set clear status during focus time
- Turn off Slack notifications
- Set expectations: "I check Slack at 12pm and 4pm"
- Create async-first culture

**"Everything feels urgent"**
- Use Eisenhower Matrix - most "urgent" is Q3
- Ask: "What happens if I don't do this today?"
- Push back: "Help me understand the urgency"
- Escalate if needed: @02-Methods-and-Tools/2.4-Communication/2.4.5-Escalation

**"I have 5 initiatives and can't focus"**
- Map context zones
- Time-block by context
- Consider: Should you have 5 initiatives?
