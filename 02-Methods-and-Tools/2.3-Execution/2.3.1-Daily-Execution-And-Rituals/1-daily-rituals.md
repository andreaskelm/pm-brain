## Daily PM Rituals

### Your Daily Operating System

**Purpose:** Create predictable structure that prevents firefighting and ensures nothing falls through cracks.

**Time Required:** 60-90 minutes of structured time across the day

-----

### MORNING ROUTINE (30-45 minutes)

**Best Time:** First thing after coffee, before meetings

**STEP 1: Check Critical Signals (10 min)**

**In this order:**

```
‚òê Production/System Health
- Check status page / monitoring dashboard
- Any incidents overnight?
- Customer-reported issues in Slack/email?

‚òê Team Blockers
- Skim team Slack channel
- Any blockers raised in async updates?
- Anyone need urgent help?

‚òê Stakeholder Requests
- New email from leadership
- Slack DMs from key stakeholders
- Any "urgent" flags

‚òê Key Metrics
- Check your core metrics dashboard
- Any unusual changes or alerts?
- Compare to yesterday/last week
```

**Decision Point:** Is anything on fire that needs immediate attention?

- **YES ‚Üí** Handle the fire (see Common Scenarios section)
- **NO ‚Üí** Continue with routine

-----

**STEP 2: Review Today's Commitments (10 min)**

**Open your calendar and task list:**

```
‚òê What meetings do I have?
- Can any be cancelled/shortened?
- What prep is needed?
- What decisions need to be made?

‚òê What must get done today?
- Commitments to others
- Deadlines
- Follow-ups I promised

‚òê What can wait if needed?
- Important but not urgent
- Can be delegated
- Can move to tomorrow
```

**Create today's focus list (3-5 items max):**

```markdown
## Today: [Date]

### MUST DO (Committed)
1. [Task with time estimate and why it matters]
2. [Task with time estimate and why it matters]

### SHOULD DO (High Value)
3. [Task with time estimate]
4. [Task with time estimate]

### COULD DO (If Time)
5. [Task with time estimate]

### DELEGATE/DEFER
- [What I'm moving or giving to others]
```

-----

**STEP 3: Protect Your Focus Time (5 min)**

**Block calendar for deep work:**

- [ ] 2-hour block for focused work (protect this)
- [ ] 30-min buffer before big meetings
- [ ] 15-min breaks between back-to-back meetings

**Set boundaries:**

```
- Slack status: "üéØ Focus time until [X]"
- Close email
- Turn off notifications
- Let team know you're heads-down
```

-----

**STEP 4: Quick Team Check-In (10 min)**

**If you have a team:**

```
‚òê Scan team updates (async standups, Slack)
‚òê Anyone blocked? Can I unblock them now?
‚òê Any celebrations or wins to acknowledge?
‚òê Anyone need air cover or support?
```

**Quick Slack message to team:**

```
Morning team! üëã

Blockers I'm handling today:
- [Blocker you're working to remove]
- [Decision you're making]

Available for:
- Quick questions: Slack me
- Longer discussions: Grab time after 2pm
- Urgent stuff: Interrupt me

Focus time: 9-11am (async only unless urgent)

Let's ship it! üöÄ
```

-----

**MORNING ROUTINE COMPLETE ‚úÖ**
You now know: What's on fire, what's due today, where you'll focus

-----

### MIDDAY CHECK-IN (10-15 minutes)

**Best Time:** Lunch or mid-afternoon (12pm-2pm)

**STEP 1: Progress Check**

```
‚òê Morning focus work: Did I complete it?
- YES ‚Üí Great, what's next priority?
- NO ‚Üí What's blocking me? Can I unblock it?

‚òê Meetings so far: Any follow-ups needed?
- Action items to capture
- Decisions to document
- People to follow up with

‚òê New issues: Anything pop up that changes priorities?
- Urgent requests
- Customer escalations
- Team blockers
```

-----

**STEP 2: Adjust Afternoon Plan**

```
Afternoon meetings: [List them]

Available time between meetings: [Calculate it]

Realistic afternoon goals:
1. [Most important thing]
2. [Second priority if time]

Moving to tomorrow:
- [What's not getting done today]
```

-----

**STEP 3: Quick Communication Sweep**

```
‚òê Slack: Respond to anything waiting on me
‚òê Email: Handle anything urgent (not full inbox zero)
‚òê Calendar: Prep for afternoon meetings
‚òê Team: Check if anyone needs help
```

-----

### EVENING WRAP-UP (15-20 minutes)

**Best Time:** Last 20 minutes of work day

**STEP 1: Close the Loop (10 min)**

```
‚òê Action items from today's meetings
- Captured in task system
- Scheduled or assigned

‚òê Promises made
- Logged and tracked
- Follow-ups scheduled

‚òê Documents/notes
- Meeting notes shared
- Decisions documented
- Key points sent to stakeholders
```

-----

**STEP 2: Prepare Tomorrow (5 min)**

```
‚òê Review tomorrow's calendar
- Any prep needed tonight?
- Any conflicts to resolve?
- Any meetings to move?

‚òê Create tomorrow's tentative focus list
- 3 most important things
- Based on what didn't get done today

‚òê Set yourself up for success
- Documents you'll need
- Questions to think about overnight
- People to talk to
```

-----

**STEP 3: Inbox/Slack Zero Attempt (5 min)**

```
‚òê Email: Get to inbox zero (or close)
- Respond to urgent
- Defer others with note
- Archive processed items

‚òê Slack: Clear DMs and mentions
- Respond or acknowledge
- Snooze to tomorrow if not urgent
- Clear notification count

‚òê Task list: Updated and prioritized
- Complete items checked off
- Tomorrow's items at top
- Overdue items addressed or rescheduled
```

-----

**STEP 4: Mental Reset (2 min)**

```
‚úÖ Wins today:
- [Something you shipped or moved forward]
- [Someone you helped]
- [Problem you solved]

üìù Learn tomorrow:
- [What could have gone better]
- [What to do differently]

üßò Let it go:
- Close computer
- Don't check Slack/email until tomorrow
- Trust your system
```
