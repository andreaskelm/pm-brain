# Decision-Making Mental Models

Quick lenses to match decision speed to risk and surface hidden assumptions.

## Use When
- You need to choose speed vs rigor (reversibility, blast radius).
- Teams disagree and you need to align on assumptions.
- You want to de-risk plans before committing.
