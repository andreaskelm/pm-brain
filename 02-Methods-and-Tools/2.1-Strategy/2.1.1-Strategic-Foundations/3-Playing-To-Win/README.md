# Playing to Win

**Framework by:** A.G. Lafley (ex-P&G CEO) & Roger Martin (Dean, Rotman School)  
**Published:** Book - Playing to Win: How Strategy Really Works (2013)  
**What it is:** Strategic choices framework via 5 cascading questions

## Files
- `playing-to-win.md` — Complete Strategic Choice Cascade methodology and guide
- `2-playing-to-win-template.md` — Ready-to-use template (Strategic Choice Cascade fill-in template)

## When to Use
- Need to make core strategic choices
- Company/product at inflection point
- Need framework for strategic debate and alignment
