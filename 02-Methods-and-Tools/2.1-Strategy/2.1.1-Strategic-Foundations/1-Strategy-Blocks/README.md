# Strategy Blocks

**Framework by:** Chandra Janakiraman (VRChat CPO, ex-Meta, Headspace)  
**Published:** [Lenny's Newsletter](https://www.lennysnewsletter.com/p/strategy-blocks-an-operators-guide)  
**What it is:** Operational framework for crafting 2-year (small "s") and 3/5/10-year (big "S") product strategy

## When to Use
- Need structured process for strategy creation (8-12 weeks)
- Want team alignment built into the process
- Need both near-term (problem-focused) and long-term (aspirational) strategy
