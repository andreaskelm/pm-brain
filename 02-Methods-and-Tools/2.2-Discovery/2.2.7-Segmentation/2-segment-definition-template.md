## Job clustering dimensions (for clustering step)

Use this section when grouping job stories into segments. For each job story, score or tag:

| Dimension | Options / scale |
|-----------|------------------|
| **Desired outcome** | Speed · Quality · Risk reduction · Status · Control · Learning · Other |
| **Job frequency** | Multiple daily · Daily · Weekly · Monthly · Rare |
| **Urgency** | Critical · Important · Nice-to-have |
| **Context** | Solo · Team · Customer-facing · Internal |
| **Primary push** | Too slow · Too complex · Missing features · Unreliable · Other |
| **Primary pull** | Automation · Simplicity · Integration · Collaboration · Other |
| **Current solution** | Manual · Competitor A/B · Nothing (non-consumption) |
| **WTP** | $0 · $1–50/mo · $50–200/mo · $200+/mo · Enterprise |

**Consolidation rule:** Combine job families into one segment if they could use ~80% same features, similar WTP band, and similar channels. Aim for 3–5 segments total.

## Example (filled): Compliance Keepers

- **Segment name:** Compliance Keepers
- **Tagline:** Regulated companies who need audit-ready documentation without slowing down operations.
- **Job:** When facing an upcoming audit or regulatory review, I want to produce complete documentation trails instantly, so I can pass audits without scrambling or delaying projects.
- **Context:** Quarterly–annually, critical urgency, team + external auditors, regulatory/time constraints.
- **Forces:** Push = scrambling for documents, missing docs cause failed audits, manual tracking error-prone. Pull = automatic audit trails, confidence nothing is missing. Anxiety = new system might not meet auditor requirements. Habit = manual checklists and spreadsheets (years).
- **Success:** Pass audit on first submission with zero findings; reduce audit prep time.
- **TAM:** ~50K companies (regulated industries). Job frequency: 1–4x per year.
- **Who:** Compliance Officer, VP Ops, Quality Manager; healthcare, finance, manufacturing; 50–5K employees. Signals: recently failed/barely passed audit, searching "audit management software."
- **Competition:** Manual spreadsheets (fired: error-prone); generic doc management (fired: no audit features); full GRC suites (fired: overkill, expensive).
- **Must-haves:** Automatic audit trail, gap analysis vs standards, auditor-friendly export.
- **Positioning:** "Never scramble for an audit again." Channels: LinkedIn, compliance conferences, consulting partnerships. Sales-assisted.
- **WTP:** $2,000–5,000/month; price-conscious (need clear ROI).
- **Validation:** Qualitative 10 interviews; quantitative 80 responses; segment validated.
- **Personas:** "Regulatory Rachel" (60%), "Quality Manager Quinn" (40%) — same segment, different roles and copy.
